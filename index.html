Airtable Settings

To set up Airtable:

  1. Create a base at airtable.com
  2. Add a table named BabyLog with fields: Type (single select), Timestamp (date time), Side (single select), StartTime (date time), Duration (number), Temperature (number)
  3. Create a token at airtable.com/create/tokens with data.records:read and data.records:write scopes
  4. Copy your Base ID from the URL (starts with app)
